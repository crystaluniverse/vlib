# openrpc

```bash
Usage: openrpc [flags] [commands]

OpenRPC Library CLI

Flags:
  -help               Prints help information.
  -version            Prints version information.
  -man                Prints the auto-generated manpage.

Commands:
  docgen              Generates docs for a given OpenRPC Client Module in V
  help                Prints help information.
  version             Prints version information.
  man                 Prints the auto-generated manpage.

```
