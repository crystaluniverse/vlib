fn get_expiration_from_timestring #

fn get_expiration_from_timestring(exp_ string) !Expiration

Get Expiration object from time string input input can be either relative or absolute

Relative time

time periods:

  • s -> second
  • h -> hour
  • d -> day
  • w -> week
  • M -> month
  • Q -> quarter
  • Y -> year 0 means right now input string example: "+1w +2d -4h"

Absolute time

inputs must be of the form: "YYYY-MM-DD HH:mm:ss" or "YYYY-MM-DD" input string examples: '2022-12-5 20:14:35' '2022-12-5' - sets hours, mins, seconds to 00 TODO: do error handling
